Домашняя страница

Описание проекта

● Это простая страница телеграмм бота, написанного на Django + Aiogram

● Через Django Admin можно добавлять и удалять категории и товары. Используется PostgreSql

● Реализована система регистрации, входа и функция "забыли пароль"

● В самом телеграмм боте можно просматривать категории и товары

● Есть также команды, такие как "помощь", "описание", "каталог", "админ"

● Команда "админ" переходит в меню администратора. В нем доступны команды только для админа, такие как "рассылка" и т.д.

● Настроена админ панель в Django

Ссылка на Django Admin

● Нажмите на меня, чтобы перейти в Django Admin, или добавь в конец URL-адреса: /admin/
